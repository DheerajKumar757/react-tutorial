To check version of node :
node -v

To check version of npm(node package manager) :
npm -v 

To update the node we need nvm which is node version manager.
To update the node to the Long Term-Support version :
nvm install --lts

You can read below page to learn about the nvm :
https://github.com/nvm-sh/nvm
